Neural Network Forward Pass using tanh activation function

This project implements a simple feedforward neural network:

2 Inputs → 2 Hidden neurons → 1 Output neuron

Specifications:
Activation Function: tanh
Weights: Random values in range [-0.5, 0.5]
Bias:
b1 = 0.5
b2 = 0.7

Run:
python tanh_network.py
